# Data Portfolio

This repository contains analytics projects built to demonstrate practical SQL, data modeling, and dashboard design skills.

## Featured Project

### Olist E-commerce Sales Analysis

Business-focused SQL analysis of the Olist marketplace dataset, designed as the analytical foundation for a Power BI dashboard.

- Project folder: [`01_olist_ecommerce`](./01_olist_ecommerce)
- Skills shown: PostgreSQL, data exploration, data quality checks, analytical view design, dashboard-ready modeling
- Current status: SQL analysis complete, Power BI dashboard in progress

Highlights:

- Validated table grain and join behavior before calculating revenue metrics
- Built reusable analytical views for order-level and order-category analysis
- Identified one missing-payment edge case and many-to-many join risk between items and payments
- Produced business-ready answers for revenue trend, top categories, and high-value states

## Project Structure

- [`01_olist_ecommerce`](./01_olist_ecommerce): SQL analysis and dashboard-ready dataset for Olist e-commerce sales
- [`02_learning_analytics`](./02_learning_analytics): second analytics project in progress

## Tools Used

The focus of this repository is not only writing queries, but showing end-to-end analytical thinking:

- translate business questions into metrics
